Installation

Just link to it! You don't have to key in a bunch of commands to install StrawberryJS, because being a front-end-only Javascript library, you can load it directly straight from a CDN. Currently, the source code can be downloaded from JSDelivr. Use the selector below to generate the link for the specific version you want to use: {{version}}

Creating an instance

To start up a new project, you will want to use the create method with the name of the application as the argument, like so: strawberry.create('app'); In our example above, we have named our project as app. We can leave the callback function empty as of now.
